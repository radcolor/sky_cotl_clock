Privacy

No Sky login. No game modification.

Isekai is a separate fan-made utility. It does not connect to your Sky account, read game data, modify the game, or need personal telemetry to work.

FAQ

Quick answers

Is Isekai official?

No. It is an unofficial fan-made project. Sky: Children of the Light belongs to thatgamecompany.

Does it work without internet?

Most app features are local, including event calculations, planner data, themes, and saved settings.

Where should I download it?

Use the download button on this page or the official GitHub releases page for this repository.
